Cloud Computing

Trabalho Cloud Computing - Unoesc

Acadêmicos: Rodrigo Angelo Valentini e Jean Scussel

Desenvolvimento do trabalho

  • Criação de uma conta gratuita na Amazon Web Services
  • Criação de uma máquina virtual (instância) com o sistema operacional ubuntu
  • Geração de chave para acesso SSH
  • Alocação de IP e associação à instância
  • Acesso à máquina via terminal para instalação do Apache
  • Criação de domínio (rodrigoangelo.tk) no Freenom
  • Adição do domínio no Cloudflare
  • Configuração de DNS no Cloudflare apontando para o IP público da instancia da Amazon
  • Alteração dos Nameservers do domínio no Freenom com os DNS obtidos no Cloudflare
  • Criação de arquivo html com o CDN do bootstrap e jquery
  • Utilização do Filezilla para conexão sftp para deploy da página so servidor
